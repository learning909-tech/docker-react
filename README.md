# Containerized React Application using Docker

## Overview
This project demonstrates containerization of a React-based application using Docker and Docker Compose, along with CI/CD integration using GitHub Actions.

## Technologies
- React.js
- Docker
- Docker Compose
- GitHub Actions

## Features
- Containerized application using Docker
- Development setup using Docker Compose
- Live code updates using volume mapping
- Automated build and test using GitHub Actions

## How to Run

### Using Docker Compose
docker-compose up

### Access Application
http://localhost:3000

## Outcome
Gained hands-on experience in containerization, Docker Compose, and integrating CI/CD using GitHub Actions.
